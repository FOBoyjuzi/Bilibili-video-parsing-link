## 一、环境准备

### 1. 服务器要求
| 项目 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 操作系统 | CentOS 7/8、Ubuntu 18.04+、Debian 9+、Windows Server 2016+ | CentOS 8 / Ubuntu 20.04 |
| CPU | 1核 | 2核 |
| 内存 | 512MB | 1GB |
| 存储空间 | 1GB | 5GB |
| 网络 | 1Mbps | 10Mbps+ |

### 2. 安装Node.js环境

#### 方式一：宝塔面板图形界面安装
1. 登录宝塔面板
2. 点击左侧菜单「软件商店」
3. 搜索「Node.js」并安装最新版（推荐 v16.x 或 v18.x）
4. 安装完成后点击「设置」-「版本管理」，确保安装了 npm

#### 方式二：Linux命令行安装（CentOS/RHEL）
```bash
# 安装 Node.js 18 LTS
curl -sL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs

# 验证安装
node -v  # 应显示 v18.x.x
npm -v   # 应显示 9.x.x 或更高
```

#### 方式三：Linux命令行安装（Ubuntu/Debian）
```bash
# 更新软件源
apt update && apt upgrade -y

# 安装 Node.js 18 LTS
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 验证安装
node -v
npm -v
```

#### 方式四：Windows服务器安装
1. 下载 Node.js 安装包：https://nodejs.org/zh-cn/download/
2. 选择 Windows Installer (.msi) 64位版本
3. 运行安装程序，按默认选项安装
4. 打开命令提示符验证：
```cmd
node -v
npm -v
```

## 二、宝塔面板部署（推荐）

### 1. 创建网站
1. 登录宝塔面板（地址：`http://your-server-ip:8888`）
2. 点击左侧菜单「网站」-「添加站点」
3. 填写信息：
   - **域名**：填写你的域名（如 `bili.example.com`），可留空使用IP访问
   - **根目录**：设置为 `/www/wwwroot/bilibili-parser`
   - **PHP版本**：选择「纯静态」（本项目无需PHP）
   - **数据库**：选择「无」（本项目无需数据库）
4. 点击「提交」

### 2. 上传项目文件
1. 点击刚创建的网站右侧「管理」按钮
2. 在弹出的菜单中选择「文件」
3. 点击「上传」-「选择文件」
4. 选择项目的以下文件进行上传：
   - `package.json`
   - `server.js`
   - `index.html`（可选，作为默认首页）
5. 上传完成后，确保文件在根目录下

### 3. 安装项目依赖
1. 在网站管理页面点击「终端」
2. 在终端中执行以下命令：
```bash
# 进入项目目录
cd /www/wwwroot/bilibili-parser

# 安装依赖（生产模式）
npm install --production

# 查看安装结果
npm ls
```

### 4. 配置PM2管理器
1. 点击左侧菜单「软件商店」-「已安装」
2. 找到「PM2管理器」并点击「设置」
3. 点击「添加项目」，填写配置：
   - **项目名称**：`bilibili-parser`（自定义名称）
   - **项目路径**：`/www/wwwroot/bilibili-parser`
   - **启动文件**：`server.js`
   - **运行用户**：`www`（宝塔默认用户）
   - **端口**：`3000`（保持与 server.js 中配置一致）
   - **环境变量**：可留空，如需修改端口可添加 `PORT=3000`
4. 点击「提交」后，再点击「启动」

### 5. 配置端口开放
1. 点击左侧菜单「安全」-「防火墙」
2. 点击「添加规则」
3. 填写：
   - **端口**：`3000`
   - **备注**：`B站视频解析服务`
   - **放行地址**：`0.0.0.0/0`（允许所有IP访问）
4. 点击「提交」

### 6. 配置反向代理（推荐使用域名访问）
1. 在网站管理页面点击「反向代理」
2. 点击「添加反向代理」
3. 填写配置：
   - **代理名称**：`bilibili-parser-proxy`
   - **目标URL**：`http://127.0.0.1:3000`
   - **发送域名**：勾选
   - **WebSocket**：勾选（用于SSE实时日志）
4. 点击「提交」

### 7. 配置SSL证书（可选但推荐）
1. 在网站管理页面点击「SSL」
2. 选择「Let's Encrypt」
3. 勾选需要证书的域名
4. 点击「申请」
5. 申请成功后，开启「强制HTTPS」

## 三、Linux服务器手动部署

### 1. 创建项目目录
```bash
# 创建网站目录
mkdir -p /www/wwwroot/bilibili-parser

# 设置目录权限
chown -R www:www /www/wwwroot/bilibili-parser
chmod -R 755 /www/wwwroot/bilibili-parser
```

### 2. 上传项目文件
```bash
# 进入项目目录
cd /www/wwwroot/bilibili-parser

# 使用 scp 从本地上传文件（替换为你的服务器IP）
scp package.json server.js index.html root@your-server-ip:/www/wwwroot/bilibili-parser/
```

### 3. 安装依赖
```bash
cd /www/wwwroot/bilibili-parser
npm install --production
```

### 4. 安装并配置PM2
```bash
# 全局安装 PM2
npm install -g pm2

# 启动服务
pm2 start server.js --name bilibili-parser

# 设置开机自启
pm2 startup
pm2 save

# 查看状态
pm2 status
```

### 5. 配置防火墙
```bash
# CentOS/RHEL 开放端口
firewall-cmd --zone=public --add-port=3000/tcp --permanent
firewall-cmd --reload

# Ubuntu/Debian 开放端口
ufw allow 3000/tcp
ufw reload
```

### 6. 配置Nginx反向代理
```bash
# 创建Nginx配置文件
cat > /etc/nginx/sites-available/bilibili-parser.conf << 'EOF'
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_read_timeout 86400;
    }
}
EOF

# 创建软链接
ln -s /etc/nginx/sites-available/bilibili-parser.conf /etc/nginx/sites-enabled/

# 测试配置并重启
nginx -t
systemctl restart nginx
```

## 四、Windows服务器部署

### 1. 创建项目目录
```cmd
mkdir C:\wwwroot\bilibili-parser
```

### 2. 上传项目文件
将 `package.json`、`server.js`、`index.html` 复制到 `C:\wwwroot\bilibili-parser`

### 3. 安装依赖
```cmd
cd C:\wwwroot\bilibili-parser
npm install --production
```

### 4. 使用PM2管理进程（推荐）
```cmd
npm install -g pm2
pm2-install
pm2 start server.js --name bilibili-parser
pm2-startup install
pm2 save
```

### 5. 配置Windows防火墙
```cmd
netsh advfirewall firewall add rule name="B站解析服务" dir=in action=allow protocol=TCP localport=3000 remoteip=any
```

## 五、Docker部署

### 1. 创建Dockerfile
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
COPY server.js .
COPY index.html .
RUN npm install --production
EXPOSE 3000
CMD ["node", "server.js"]
```

### 2. 构建并运行容器
```bash
docker build -t bilibili-parser .
docker run -d --name bilibili-parser -p 3000:3000 -v $(pwd)/access.log:/app/access.log bilibili-parser
```

## 七、常见问题及解决

### 1. 502 Bad Gateway 错误
**解决方法：**
```bash
pm2 status
netstat -tulpn | grep 3000
firewall-cmd --list-ports
```

### 2. 403 Forbidden 错误
**解决方法：**
```bash
chown -R www:www /www/wwwroot/bilibili-parser
chmod -R 755 /www/wwwroot/bilibili-parser
```

### 3. 服务自动停止
**解决方法：**
```bash
pm2 startup
pm2 save
```
