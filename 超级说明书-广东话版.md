## 一、環境準備

### 1. 服務器要求
| 項目 | 最低要求 | 推薦配置 |
|------|---------|---------|
| 操作系統 | CentOS 7/8、Ubuntu 18.04+、Debian 9+、Windows Server 2016+ | CentOS 8 / Ubuntu 20.04 |
| CPU | 1核 | 2核 |
| 內存 | 512MB | 1GB |
| 存儲空間 | 1GB | 5GB |
| 網絡 | 1Mbps | 10Mbps+ |

### 2. 安裝Node.js環境

#### 方式一：寶塔面板圖形界面安裝
1. 登錄寶塔面板
2. 點擊左側菜單「軟件商店」
3. 搜索「Node.js」並安裝最新版（推薦 v16.x 或 v18.x）
4. 安裝完成後點擊「設置」-「版本管理」，確保安裝了 npm

#### 方式二：Linux命令行安裝（CentOS/RHEL）
```bash
# 安裝 Node.js 18 LTS
curl -sL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs

# 驗證安裝
node -v  # 應顯示 v18.x.x
npm -v   # 應顯示 9.x.x 或更高
```

#### 方式三：Linux命令行安裝（Ubuntu/Debian）
```bash
# 更新軟件源
apt update && apt upgrade -y

# 安裝 Node.js 18 LTS
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 驗證安裝
node -v
npm -v
```

#### 方式四：Windows服務器安裝
1. 下載 Node.js 安裝包：https://nodejs.org/zh-cn/download/
2. 選擇 Windows Installer (.msi) 64位版本
3. 運行安裝程序，按默認選項安裝
4. 打開命令提示符驗證：
```cmd
node -v
npm -v
```

## 二、寶塔面板部署（推薦）

### 1. 創建網站
1. 登錄寶塔面板（地址：`http://your-server-ip:8888`）
2. 點擊左側菜單「網站」-「添加站點」
3. 填寫信息：
   - **域名**：填寫你的域名（如 `bili.example.com`），可留空使用IP訪問
   - **根目錄**：設置為 `/www/wwwroot/bilibili-parser`
   - **PHP版本**：選擇「純靜態」（本項目無需PHP）
   - **數據庫**：選擇「無」（本項目無需數據庫）
4. 點擊「提交」

### 2. 上傳項目文件
1. 點擊剛創建的網站右側「管理」按鈕
2. 在彈出的菜單中選擇「文件」
3. 點擊「上傳」-「選擇文件」
4. 選擇項目的以下文件進行上傳：
   - `package.json`
   - `server.js`
   - `index.html`（可選，作為默認首頁）
5. 上傳完成後，確保文件在根目錄下

### 3. 安裝項目依賴
1. 在網站管理頁面點擊「終端」
2. 在終端中執行以下命令：
```bash
# 進入項目目錄
cd /www/wwwroot/bilibili-parser

# 安裝依賴（生產模式）
npm install --production

# 查看安裝結果
npm ls
```

### 4. 配置PM2管理器
1. 點擊左側菜單「軟件商店」-「已安裝」
2. 找到「PM2管理器」並點擊「設置」
3. 點擊「添加項目」，填寫配置：
   - **項目名稱**：`bilibili-parser`（自定義名稱）
   - **項目路徑**：`/www/wwwroot/bilibili-parser`
   - **啟動文件**：`server.js`
   - **運行用戶**：`www`（寶塔默認用戶）
   - **端口**：`3000`（保持與 server.js 中配置一致）
   - **環境變量**：可留空，如需修改端口可添加 `PORT=3000`
4. 點擊「提交」後，再點擊「啟動」

### 5. 配置端口開放
1. 點擊左側菜單「安全」-「防火牆」
2. 點擊「添加規則」
3. 填寫：
   - **端口**：`3000`
   - **備註**：`B站視頻解析服務`
   - **放行地址**：`0.0.0.0/0`（允許所有IP訪問）
4. 點擊「提交」

### 6. 配置反向代理（推薦使用域名訪問）
1. 在網站管理頁面點擊「反向代理」
2. 點擊「添加反向代理」
3. 填寫配置：
   - **代理名稱**：`bilibili-parser-proxy`
   - **目標URL**：`http://127.0.0.1:3000`
   - **發送域名**：勾選
   - **WebSocket**：勾選（用於SSE實時日誌）
4. 點擊「提交」

### 7. 配置SSL證書（可選但推薦）
1. 在網站管理頁面點擊「SSL」
2. 選擇「Let's Encrypt」
3. 勾選需要證書的域名
4. 點擊「申請」
5. 申請成功後，開啟「強制HTTPS」

## 三、Linux服務器手動部署

### 1. 創建項目目錄
```bash
# 創建網站目錄
mkdir -p /www/wwwroot/bilibili-parser

# 設置目錄權限
chown -R www:www /www/wwwroot/bilibili-parser
chmod -R 755 /www/wwwroot/bilibili-parser
```

### 2. 上傳項目文件
```bash
# 進入項目目錄
cd /www/wwwroot/bilibili-parser

# 使用 scp 從本地上傳文件（替換為你的服務器IP）
scp package.json server.js index.html root@your-server-ip:/www/wwwroot/bilibili-parser/
```

### 3. 安裝依賴
```bash
cd /www/wwwroot/bilibili-parser
npm install --production
```

### 4. 安裝並配置PM2
```bash
# 全局安裝 PM2
npm install -g pm2

# 啟動服務
pm2 start server.js --name bilibili-parser

# 設置開機自啟
pm2 startup
pm2 save

# 查看狀態
pm2 status
```

### 5. 配置防火牆
```bash
# CentOS/RHEL 開放端口
firewall-cmd --zone=public --add-port=3000/tcp --permanent
firewall-cmd --reload

# Ubuntu/Debian 開放端口
ufw allow 3000/tcp
ufw reload
```

### 6. 配置Nginx反向代理
```bash
# 創建Nginx配置文件
cat > /etc/nginx/sites-available/bilibili-parser.conf << 'EOF'
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_read_timeout 86400;
    }
}
EOF

# 創建軟鏈接
ln -s /etc/nginx/sites-available/bilibili-parser.conf /etc/nginx/sites-enabled/

# 測試配置並重啟
nginx -t
systemctl restart nginx
```

```cmd
npm install -g pm2
pm2-install

```

```cmd
netsh advfirewall firewall add rule name="B站解析服務" dir=in action=allow protocol=TCP localport=3000 remoteip=any
```

## 五、Docker部署

### 1. 創建Dockerfile
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
COPY server.js .
COPY index.html .
RUN npm install --production
EXPOSE 3000
CMD ["node", "server.js"]
```

### 2. 構建並運行容器
```bash
docker build -t bilibili-parser .
docker run -d --name bilibili-parser -p 3000:3000 -v $(pwd)/access.log:/app/access.log bilibili-parser
```

## 七、常見問題及解決

### 1. 502 Bad Gateway 錯誤
**解決方法：**
```bash
pm2 status
netstat -tulpn | grep 3000
firewall-cmd --list-ports
```

### 2. 403 Forbidden 錯誤
**解決方法：**
```bash
chown -R www:www /www/wwwroot/bilibili-parser
chmod -R 755 /www/wwwroot/bilibili-parser
```

### 3. 服務自動停止
**解決方法：**
```bash
pm2 startup
pm2 save
```
