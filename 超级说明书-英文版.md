## 1. Environment Preparation

### 1. Server Requirements
| Item | Minimum | Recommended |
|------|---------|-------------|
| OS | CentOS 7/8, Ubuntu 18.04+, Debian 9+, Windows Server 2016+ | CentOS 8 / Ubuntu 20.04 |
| CPU | 1 Core | 2 Cores |
| Memory | 512MB | 1GB |
| Storage | 1GB | 5GB |
| Network | 1Mbps | 10Mbps+ |

### 2. Install Node.js

#### Method 1: BT Panel (Recommended)
1. Login to BT Panel
2. Click "Software Store" in left menu
3. Search for "Node.js" and install latest version (v16.x or v18.x recommended)
4. After installation, click "Settings" -> "Version Management" to ensure npm is installed

#### Method 2: Linux CLI (CentOS/RHEL)
```bash
# Install Node.js 18 LTS
curl -sL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs

# Verify installation
node -v  # Should show v18.x.x
npm -v   # Should show 9.x.x or higher
```

#### Method 3: Linux CLI (Ubuntu/Debian)
```bash
# Update packages
apt update && apt upgrade -y

# Install Node.js 18 LTS
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# Verify installation
node -v
npm -v
```

#### Method 4: Windows Server
1. Download Node.js installer: https://nodejs.org/en/download/
2. Select Windows Installer (.msi) 64-bit
3. Run installer with default options
4. Verify in Command Prompt:
```cmd
node -v
npm -v
```

## 2. BT Panel Deployment (Recommended)

### 1. Create Website
1. Login to BT Panel (`http://your-server-ip:8888`)
2. Click "Website" -> "Add Site" in left menu
3. Fill in information:
   - **Domain**: Your domain (e.g., `bili.example.com`), or leave blank for IP access
   - **Root Directory**: `/www/wwwroot/bilibili-parser`
   - **PHP Version**: Select "Pure Static" (PHP not required)
   - **Database**: Select "None" (Database not required)
4. Click "Submit"

### 2. Upload Files
1. Click "Manage" button next to the created site
2. Select "Files" from the menu
3. Click "Upload" -> "Select Files"
4. Upload these project files:
   - `package.json`
   - `server.js`
   - `index.html` (optional, as default homepage)
5. Ensure files are in root directory

### 3. Install Dependencies
1. Click "Terminal" in website management
2. Execute commands:
```bash
cd /www/wwwroot/bilibili-parser
npm install --production
npm ls
```

### 4. Configure PM2
1. Click "Software Store" -> "Installed"
2. Find "PM2 Manager" and click "Settings"
3. Click "Add Project" with configuration:
   - **Project Name**: `bilibili-parser`
   - **Project Path**: `/www/wwwroot/bilibili-parser`
   - **Startup File**: `server.js`
   - **Run User**: `www`
   - **Port**: `3000`
   - **Environment Variables**: Leave blank or add `PORT=3000`
4. Click "Submit", then "Start"

### 5. Open Port
1. Click "Security" -> "Firewall" in left menu
2. Click "Add Rule"
3. Fill in:
   - **Port**: `3000`
   - **Remark**: `Bilibili Parser Service`
   - **Allow Address**: `0.0.0.0/0`
4. Click "Submit"

### 6. Configure Reverse Proxy
1. Click "Reverse Proxy" in website management
2. Click "Add Reverse Proxy"
3. Fill in:
   - **Proxy Name**: `bilibili-parser-proxy`
   - **Target URL**: `http://127.0.0.1:3000`
   - **Send Domain**: Checked
   - **WebSocket**: Checked (for SSE real-time logs)
4. Click "Submit"

### 7. Configure SSL (Optional but Recommended)
1. Click "SSL" in website management
2. Select "Let's Encrypt"
3. Check domains for certificate
4. Click "Apply"
5. Enable "Force HTTPS" after success

## 3. Linux Manual Deployment

### 1. Create Directory
```bash
mkdir -p /www/wwwroot/bilibili-parser
chown -R www:www /www/wwwroot/bilibili-parser
chmod -R 755 /www/wwwroot/bilibili-parser
```

### 2. Upload Files
```bash
cd /www/wwwroot/bilibili-parser
scp package.json server.js index.html root@your-server-ip:/www/wwwroot/bilibili-parser/
```

### 3. Install Dependencies
```bash
cd /www/wwwroot/bilibili-parser
npm install --production
```

### 4. Install PM2
```bash
npm install -g pm2
pm2 start server.js --name bilibili-parser
pm2 startup
pm2 save
```

### 5. Configure Firewall
```bash
# CentOS/RHEL
firewall-cmd --zone=public --add-port=3000/tcp --permanent
firewall-cmd --reload

# Ubuntu/Debian
ufw allow 3000/tcp
ufw reload
```

### 6. Configure Nginx
```bash
cat > /etc/nginx/sites-available/bilibili-parser.conf << 'EOF'
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_read_timeout 86400;
    }
}
EOF

ln -s /etc/nginx/sites-available/bilibili-parser.conf /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

## 4. Windows Deployment

### 1. Create Directory
```cmd
mkdir C:\wwwroot\bilibili-parser
```

### 2. Upload Files
Copy `package.json`, `server.js`, `index.html` to `C:\wwwroot\bilibili-parser`

### 3. Install Dependencies
```cmd
cd C:\wwwroot\bilibili-parser
npm install --production
```

### 4. Configure PM2
```cmd
npm install -g pm2
pm2-install
pm2 start server.js --name bilibili-parser
pm2-startup install
pm2 save
```

### 5. Configure Firewall
```cmd
netsh advfirewall firewall add rule name="Bilibili Parser" dir=in action=allow protocol=TCP localport=3000 remoteip=any
```

## 5. Docker Deployment

### 1. Create Dockerfile
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
COPY server.js .
COPY index.html .
RUN npm install --production
EXPOSE 3000
CMD ["node", "server.js"]
```

### 2. Build and Run
```bash
docker build -t bilibili-parser .
docker run -d --name bilibili-parser -p 3000:3000 -v $(pwd)/access.log:/app/access.log bilibili-parser
```
